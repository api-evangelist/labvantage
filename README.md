# LabVantage Solutions (labvantage)

LabVantage Solutions provides an end-to-end laboratory informatics platform - LIMS (Laboratory Information Management System), ELN (Electronic Lab Notebook), LES (Laboratory Execution System), and SDMS (Scientific Data Management System) - with APIs for GxP-compliant data exchange in pharmaceutical, biotech, and clinical laboratory environments.
